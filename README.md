# 仮設Webページ README

このリポジトリは、岐阜大学の学園祭「岐阜大学祭」の一時的なWebページのソースコードを管理しています。

## 構成

- `index.html` : メインページ
- `style.css` : スタイルシート
- `script.js` : JavaScriptファイル

## セットアップ

1. このリポジトリをクローンします。
2. 必要に応じてファイルを編集します。
3. `index.html` をブラウザで開いて動作を確認します。

## 開発

- 変更後はブラウザでリロードして確認してください。
- 不明点や改善案があればIssueまたはPull Requestでご連絡ください。
